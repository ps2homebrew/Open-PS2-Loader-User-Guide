Theme gallery¶

This gallery features the themes hosted at PSX-scene. Few words :

  • When they were several versions of the same theme, I only included the last version ;

  • I did not include all N-Joy rip-offs… damn, this theme was raped by the www… :/ ;

  • I did not include template themes posted by devs for learning purpose ;

  • Some themes – prior to 2012 – are just lost :/ ;

  • xxx (yyy) means that the theme is a port, where xx is the autor of the port and yyy the original author ;

  • If you want to be featured in the gallery, keep posting your themes at PSX-scene thread, I will include them here.

  • All credits to authors of these themes.
