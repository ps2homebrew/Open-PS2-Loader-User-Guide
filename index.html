Open PS2 Loader – User Guide¶

Warning : this is not OPL official repository, only a user guide for it. If you are interested in OPL src code, you can find it here

About Open PS2 Loader :¶

Open PS2 Loader (OPL) is a 100% Open source game loader for the PS2 and PS3 units. Open PS2 Loader can load games though USB, network (SMB) and from the PS2’s HDD unit. It supports all models of the PlayStation 2, and the PlayStation 3 models with PlayStation 2 backward compatibility.
